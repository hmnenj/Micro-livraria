# Micro Ecommerce

## Como executar
1. **Instale as dependências:**

```bash
npm install
```

2. **Inicie o serviço de inventário:**

```bash
node services/inventory/index.js
```

3. **Inicie o serviço de frete:**

```bash
node services/shipping/index.js
```

4. **Inicie a aplicação principal:**

```bash
npm run start
```
